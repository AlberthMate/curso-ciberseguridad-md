# 🛡️ Cybersecurity Journey: De Zero a Blue Team Hero
## Proceso de Formación Profesional en Ciberseguridad

![Estado del Proyecto](https://img.shields.io/badge/Estado-En%20Progreso-green)
![Rol Objetivo](https://img.shields.io/badge/Objetivo-Analista%20SOC%20/%20Red%20Teamer-blue)

Este repositorio contiene mi progreso detallado, laboratorios prácticos y documentación técnica siguiendo la metodología del **Profesor Cyber-X**. Mi objetivo es dominar las bases de redes, seguridad ofensiva y defensa activa para ser empleable en el sector de la ciberseguridad en un plazo de 12-18 meses.

---

## 🚀 Hoja de Ruta (Roadmap)

### Fase 01: Cimientos Técnicos (Actual 🛠️)
- [ ] **Redes y Protocolos:** Modelo OSI, TCP/IP, DNS, HTTP/S.
- [ ] **Sistemas Operativos:** Hardening de Linux y Windows.
- [ ] **Seguridad de Red:** Firewalls, IDS/IPS y VPNs.

### Fase 02: Operaciones Defensivas (Blue Team 🛡️)
- [ ] Análisis de Tráfico (Wireshark/Zeek).
- [ ] Gestión de Logs y SIEM (Splunk/ELK).
- [ ] Respuesta ante Incidentes.

### Fase 03: Seguridad Ofensiva (Red Team ⚔️)
- [ ] Pentesting Web (OWASP Top 10).
- [ ] Explotación de Redes.
- [ ] Scripting para Ciberseguridad (Python/Bash).
